# 鲁班智考充值与计费规则

更新时间：2026-04-21

## 1. 目标

本规则用于统一鲁班智考小程序的 C 端充值、点数消耗和毛利边界。

当前硬约束：

- 标准对话直接成本：`0.2 元 / 次`
- 目标毛利：`>= 70%`
- 当前对外展示充值档位：`9 / 99 / 199`

## 2. 定价底线

若标准对话直接成本为 `0.2 元`，要保证 `70%` 毛利，则：

`单次售价下限 = 0.2 / (1 - 0.7) = 0.667 元`

这意味着：

- 所有套餐在“用户把额度用满”的情况下，折算到每次标准对话的收入都必须高于 `0.667 元`
- 实际定价应高于理论下限，以覆盖支付通道费、活动优惠、退款损耗和客服成本

当前采用的设计目标是把标准对话有效单价控制在：

- `0.75 ~ 0.90 元 / 次`

## 3. 点数体系

前台不直接卖“对话次数”，统一卖“智力点”。

基础换算：

- `1 次标准问答 = 10 点`

这样做的目的：

- 用户理解简单
- 页面价格稳定
- 后台可以按任务复杂度灵活扣点
- 高成本能力不会和普通问答混成一个价

## 4. 正式充值套餐

当前推荐正式上线版本：

- `9 元 = 100 点`
- `99 元 = 1200 点`
- `199 元 = 2600 点`

对外命名：

- `9 元`：轻量体验
- `99 元`：进阶主力
- `199 元`：冲刺强化

## 5. 单位经济测算

按 `10 点 = 1 次标准问答` 计算：

- `9 元 = 10 次标准问答`
- `99 元 = 120 次标准问答`
- `199 元 = 260 次标准问答`

折算结果：

- `9 元档`：`0.90 元 / 次`，毛利约 `77.8%`
- `99 元档`：`0.825 元 / 次`，毛利约 `75.8%`
- `199 元档`：`0.765 元 / 次`，毛利约 `73.9%`

结论：

- 三档均高于 `70%` 毛利线
- 中档最适合做主力档
- 大档给到合理优惠，但不会把利润打穿

## 6. 点数消耗规则

建议按任务复杂度分层扣点：

- `快速答疑 = 8 点`
  - 适用：概念解释、术语查询、简单追问
- `标准问答 = 10 点`
  - 适用：一般题目讲解、知识点问答
- `深度解析 = 20 点`
  - 适用：分步解题、错因分析、规范对比
- `图片 / 长文档 / 复杂批改 = 30 点`
  - 适用：图题识别、材料解析、长文本批改
- `学习规划 / 学情报告 = 40 点`
  - 适用：阶段诊断、复习路径、个性化建议

产品原则：

- 普通问答按 `8~10 点` 扣
- 高成本任务必须显著高于普通问答
- 后台可根据模型、上下文长度、工具调用次数做自动分档

## 7. 页面展示原则

充值页不强调“每点多少钱”，强调“这档适合什么人”。

推荐展示文案：

- `轻量体验`
  - `100 点`
  - `适合先体验答疑、解析和日常提问`
- `进阶主力`
  - `1200 点`
  - `适合大多数备考阶段，高频问答和复盘更从容`
  - `推荐`
- `冲刺强化`
  - `2600 点`
  - `适合考前冲刺、密集刷题和深度推理`
  - `冲刺`

配套说明文案：

`智力点用于 AI 答疑、深度解析、图片识别和学习规划，不同任务按复杂度消耗不同点数。普通问答通常消耗 8-10 点。`

## 8. 不建议的方案

当前不建议采用下面这种过于激进的大档赠送：

- `99 元 = 1500 点`
- `199 元 = 3600 点`

原因：

- `99 元 = 150 次标准问答`，折算约 `0.66 元 / 次`
- `199 元 = 360 次标准问答`，折算约 `0.553 元 / 次`

这会导致：

- 中高档位逼近或跌破 `70%` 毛利线
- 最重度、最忠诚的用户反而成为利润最差的人群

## 9. 后续会员策略

当前阶段不建议主推“无限问”月卡。

如后续需要会员方案，建议做成：

- `会员 = 固定点数 + 优先级 + 附加权益`
- 超出会员内额度后继续扣点

可作为下一阶段备选方案：

- `39~49 元 / 月`
- 每月送固定额度点数
- 赠送学情报告、优先排队、会员标识等权益
- 不提供真无限

## 10. 拍板结论

若今天直接上线，推荐执行以下版本：

- 充值档位：`9 / 99 / 199`
- 套餐点数：`100 / 1200 / 2600`
- 扣点规则：`8 / 10 / 20 / 30 / 40`
- 默认主推：`99 元进阶主力`

这套方案同时满足：

- 用户能看懂
- 页面好卖
- 中档主力明显
- 大档不亏钱
- 整体毛利稳在 `70%+`
