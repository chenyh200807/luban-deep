# 选择题讲解

适用于：单选题、多选题、判断题、带 A/B/C/D 选项的题目讲解。

## 目标

- 先拿到正确答案
- 再看清每个干扰项为什么错
- 最终形成稳定判题标准

## 输出要求

- 默认结论先行
- 必须逐项判断，至少说明：
  - 正确项为什么对
  - 最易误选项为什么错
- 若题干考数值、门槛、程序顺序，优先引用知识库证据
- “易错点”要明确写出：
  - 最容易混的点
  - 错误答案看起来合理的原因
  - 本题真正的判定关键词

## FAST 约束

- 先给答案，再做 2-4 句高密度解析
- 至少保留“踩分点 / 易错点”
- 口诀只在确有助记价值时给

## DEEP 约束

- 除答案外，补全四要素
- 最后给一句迁移规则：下次遇到同类题先看什么
